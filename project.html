Weather App

  • Overview:

    The Weather App is a web-based application that allows users to view real-time weather information for any city or their current location. It fetches data from a public weather API and displays it in a clean, user-friendly interface.

  • Key Features:
    • Search weather by city name
    • Display temperature, humidity, wind speed, and general weather conditions
    • Dynamic weather icons based on conditions
    • Optionally use geolocation to auto-detect current location
    • Responsive design for both desktop and mobile

  • Tech Stack:
    • HTML/CSS for structure and styling
    • JavaScript for functionality and API calls
    • OpenWeatherMap API for real-time weather data

Calculator App

  • Overview:

    The Calculator App is a simple yet fully functional web calculator that performs basic arithmetic operations. It helps users quickly calculate results using a sleek and interactive interface.

  • Key Features:
    • Addition, subtraction, multiplication, and division
    • Clear (C) button to reset input
    • Support for decimals
    • Displays current operation and result in real-time
    • Clickable buttons with hover effects

  • Tech Stack:
    • HTML/CSS for layout and design
    • JavaScript for core logic and event handling
